• User Class
    User.all()
    Returns all Users
  • User Object
    user.is_authenticated()
    Returns True if User is logged in
    user.is_anonymous()
    Returns False if User is logged in
    user.is_active()
    Returns True if the User account is active
    user.get_id()
    Returns the User's id
    user.get_roles()
    Returns a list of the User's Roles
    user.get_display_name()
    Returns either first name, full name, or alias of the User depending on what they have selected to show in their profile, default is alias (default alias is the users email without '@domain.tdl')
    user.get_user_by_email(email)
    Returns User
    user.is_admin()
    Returns True if User has admin Role
    user.has_role(role)
    Return True if User has role in Roles
  • In the view:
    current_user
    this is the current user, check if they are logged in: current_user.is_authenticated()
